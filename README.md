# ojstat
A Crawler to get student's grade on NCCU Zerojudge : http://judge.nccucs.org/ShowProblem 

# Use

Open your terminal:

	git clone https://github.com/rootsofworld/ojstat.git
	cd ojstat
	npm install

then

    node ojstat.js problemId username password
